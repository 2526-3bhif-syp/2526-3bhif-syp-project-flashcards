== Description

Flashcards helps you learn with digital cards. It is easy to use and works on all devices. You can learn anywhere.

Key features:

* **Easy Learning**: Learn quickly with flashcards.
* **Simple Design**: Easy to use for everyone.
* **Everywhere**: Works on PC, tablet and phone.
